## Q: What are Functions in JS? What are the types of function?
Ans: A function is a **reusable block of code** that performs a specific task.
```js
function add(a, b){
    return a+b;
}
result= add(3, 5);
console.log(result)//8
```
```js
function display(){
    a = 9;
    console.log(a); // 9
}
let a;
display();
//let a;// Cannot access 'a' before initialization
```
